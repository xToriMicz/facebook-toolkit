Login / Logout

1
กดปุ่ม Login with Facebook ที่หน้าแรก
2
อนุญาตสิทธิ์: จัดการเพจ, อ่านข้อมูลเพจ, โพสเนื้อหา
3
ระบบจะดึงรายชื่อเพจที่คุณดูแล — เลือกเพจที่ต้องการ
4
Logout: กดรูปโปรไฟล์มุมขวาบน > Logout

Token จะถูกเก็บอย่างปลอดภัยในระบบ ไม่ส่งให้บุคคลที่สาม

เขียนโพส

โพสข้อความ

เขียนข้อความในช่อง แล้วกด โพสเลย — โพสลงเพจที่เลือกทันที

โพสรูป

คลิกพื้นที่ upload หรือลากรูปมาวาง รองรับหลายรูปพร้อมกัน (multi-photo post)

AI Writer

เลือก tone (ทั่วไป/ให้ความรู้) แล้วกด AI เขียนให้ — AI จะเขียนข้อความพร้อม hashtag

แม่แบบ

เลือกแม่แบบจาก dropdown — ข้อความจะใส่ใน textarea ให้แก้ไขก่อนโพส

ประวัติโพส

Filter ตามเพจ

เลือกเพจจาก dropdown ด้านบน — แสดงเฉพาะโพสของเพจนั้น

Engagement Data

แต่ละโพสแสดง likes, comments, shares ดึงจาก Facebook Graph API จริง

Comments

กด comments เพื่อดูและตอบคอมเมนต์จากหน้า dashboard

ตั้งเวลา / Bulk Schedule

ตั้งเวลาโพสเดี่ยว

1
เขียนข้อความ
2
เลือกวันที่ + เวลา
3
กด ตั้งเวลาโพส

ระบบ Cron ตรวจทุก 5 นาที — ถึงเวลาโพสอัตโนมัติ

Bulk Schedule

เลือกฉบับร่างหลายชิ้น > เลือกเวลาเริ่ม + ช่วงห่าง (1/2/3/6/12/24 ชม.) > ตั้งเวลาทั้งหมดพร้อมกัน

ฉบับร่าง

  • บันทึก — กด Save Draft ที่หน้าเขียนโพส
  • แก้ไข — กดปุ่มแก้ไข โหลดข้อความกลับ textarea
  • โพสเลย — กดปุ่มโพส draft ลงเพจทันที
  • ลบ — กดปุ่มลบ

Insights สถิติเพจ

Post Performance

Top posts / Worst posts ranked by engagement (likes+comments+shares)

Best Time to Post

วิเคราะห์จากโพสที่ผ่านมา — แนะนำวันและเวลาที่ได้ engagement สูงสุด

Refresh

กด Refresh เพื่อดึง engagement ล่าสุดจาก Facebook (20 โพสล่าสุด)

แจ้งบัค / ขอฟีเจอร์

1
เลือกประเภท: Bug / Feature / Question
2
ใส่หัวข้อ + รายละเอียด
3
แนบ URL รูปประกอบ (optional)
4
กด ส่ง Ticket

ติดตามสถานะ ticket ได้ที่รายการด้านล่าง (open / in_progress / closed)

RSS Auto-post

1
เพิ่ม RSS URL (เช่น workpointtoday.com/feed/)
2
กด Fetch — ดึงข่าวล่าสุด
3
ระบบสร้าง Draft อัตโนมัติจากข่าวใหม่ล่าสุด

Auto-draft: เมื่อมีข่าวใหม่ที่ยังไม่เคยดึง ระบบสร้าง draft ให้ตรวจสอบก่อนโพส

Affiliate Mode

เปิด toggle Affiliate ที่หน้าเขียนโพส — เพิ่ม #ad อัตโนมัติ + แนะนำใส่ link ใน comment แรก

อ่านคู่มือละเอียด: คู่มือ Affiliate

AI Writer

Provider

รองรับ Anthropic (Claude), OpenAI (GPT), Google (Gemini), Custom (OpenAI-compatible)

ตั้งค่า

ไปที่ AI Settings > เลือก provider > ใส่ API key > เลือก model > Test

ใช้งาน

เลือก tone ที่ต้องการ > กด AI เขียนให้ > AI สร้างข้อความ + hashtag
