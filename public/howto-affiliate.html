1. Affiliate Mode คืออะไร?

Affiliate Mode ช่วยให้คุณโพสเนื้อหาพร้อม affiliate link ได้อย่างถูกกฎหมายและไม่ถูก Facebook ลด reach โดยระบบจะ:

  • เพิ่ม #ad #affiliate อัตโนมัติ ตามกฎ FTC/Facebook
  • แนะนำให้ใส่ link ใน comment แรกแทนใส่ในโพส
  • แสดง warning เรื่อง reach ที่อาจลดลง

2. วิธีใช้งาน

1
เปิด Affiliate Mode — ติ๊กเครื่องหมาย 🏷️ Affiliate ที่หน้าเขียนโพส
2
เขียนเนื้อหา — เขียนรีวิว/แนะนำสินค้าอย่างจริงใจ ระบบจะเพิ่ม #ad #affiliate ท้ายโพสอัตโนมัติ
3
ใส่ affiliate link — กรอก URL ในช่อง "ลิงค์สินค้า affiliate"
4
โพส — ระบบจะโพสเนื้อหา และใส่ link ใน comment แรกให้อัตโนมัติ (ถ้าเปิดตัวเลือกนี้)

3. เทคนิคเพิ่ม Reach

Link ใน Comment แรก

98% ของโพสที่ได้ reach สูงบน Facebook ไม่มี external link ในตัวโพส ใส่ link ใน comment แรกแทนเพื่อไม่ถูกลด reach

เขียนแบบ PAS (Problem-Agitate-Solution)

เริ่มด้วยปัญหาที่คนเจอ → กระตุ้นให้รู้สึก → แนะนำทางออก (สินค้า)

ใช้รูปภาพจริง

รูปถ่ายสินค้าจริง ได้ engagement สูงกว่ารูป stock 3-5 เท่า

4. ตัวอย่างโพส Affiliate ที่ดี

ลองใช้ [ชื่อสินค้า] มา 2 เดือน — ผลลัพธ์เกินคาด! ก่อนหน้านี้มีปัญหา [ปัญหา] มาตลอด ลองหลายวิธีแล้วไม่ได้ผล จนเพื่อนแนะนำให้ลอง [สินค้า] สิ่งที่ชอบ: - [จุดเด่น 1] - [จุดเด่น 2] - [จุดเด่น 3] ใครสนใจดูรายละเอียดในคอมเมนต์แรกนะ #รีวิว #ad #affiliate

5. สิ่งที่ต้องระวัง

ข้อห้าม Facebook
  • ห้ามโพส affiliate link เกิน 5 ครั้ง/วัน → อาจถูก restrict
  • ห้ามใช้ engagement bait เช่น "กดไลค์ถ้าเห็นด้วย"
  • ห้ามโพสเนื้อหาหลอกลวง/เกินจริง
  • ต้อง disclose (#ad) ทุกครั้งที่ได้ค่าคอมมิชชัน
Reach ที่อาจลดลง

Facebook algorithm ลด reach โพสที่มี external link ถ้าต้องใส่ link ในโพส ให้เขียนเนื้อหาที่มีคุณค่าก่อน อย่าขายของตรงๆ

6. Content Formula แนะนำ

PAS (Problem-Agitate-Solution)

P: "เคยมีปัญหา [X] ไหม?"

A: "ยิ่งปล่อยไว้ยิ่ง [ผลเสีย]..."

S: "ลอง [สินค้า] แล้วชีวิตเปลี่ยน"

BAB (Before-After-Bridge)

Before: "ก่อนหน้านี้ [สถานะเดิม]"

After: "ตอนนี้ [ผลลัพธ์ที่ดีขึ้น]"

Bridge: "[สินค้า] คือตัวเปลี่ยน"
